Pour télécharger mon application Androïd
Dernière modif. de l'application: 19/02/2024

Comment l'installer ?
  1. Cliquer sur le lien ci-dessus afin de télécharger le fichier .apk
  2. Si ce n'est pas déjà fait, aller dans les paramètres et autoriser l'installation des applications de source inconnue
  3. Aller dans l'application Mes fichiers et cliquer sur le .apk téléchargé
  4. Installer l'application

Pour l'explication, l'installation est lourde car mon application n'est pas disponible sur le Playstore donc le téléphone sait qu'il n'y a pas eu de contrôle de sécurité, ce qui implique que techniquement vous prenez un risque en téléchargeant une application comme cela. Je ne cherche pas à publier mon application car c'est lourd et potentiellement source de problèmes pour moi.
