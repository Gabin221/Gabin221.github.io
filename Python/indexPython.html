.py en .exe


La création d'un fichier .exe à partir d'un .py se fait en utilisant Pyinstaller. C'est juste un module Python. On fait donc ceci en utilisant l'invite de commande et en suivant les instructions ci-dessous :


  1. pip install pyinstaller (cette étape n'est sans doutes pas obligatoire)
  2. se déplacer dans le dossier qui contient le .py à convertir
  3. pyinstaller --one file NOMDUFICHIER.py




Afficher les n premiers nombres


L'idée derrière ce code est d'afficher les n premiers nombres. Pour se faire, on va utiliser une boucle for qui va de 0 à n (ici n=3) et qui les affichent à l'écran.

    

# Le code Python

for i in range(0, 3):
    print(i)




Tracer un graphe avec Matplotlib

    

# Le code Python

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-5, 10, 1000)
y = x**2

plt.figure()
plt.plot(x, y, label="f(x)")
plt.xlim(-5, 10)
plt.ylim()
plt.grid()
plt.legend()
plt.show()




Calculer pi

    

# Le code Python

import numpy as np


def calculPi():
    n = 100000
    compteur = 0
    for i in range(n):
        x = np.random.random()
        y = np.random.random()
        if x**2 + y**2 <= 1:
            compteur += 1
    return compteur*4/n


nbrMoyenne = 30
pi = np.empty(nbrMoyenne)
for i in range(0, nbrMoyenne):
    pi[i] = calculPi()

print(f"pi = {np.mean(pi)} ± {np.abs(np.mean(pi) - np.pi)/2}")
