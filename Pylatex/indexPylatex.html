Pylatex est un package python permettant de générer facilement un fichier PDF via LaTeX. Générer un fichier LaTeX via ce procéder présente l'avantage d'avoir a priori moins de lignes de codes. De plus, l'avantage le plus important c'est que pour générer des calculs dans un document LaTeX, la méthode traditionnelle consiste à faire le calcul à la main puis le recopier dans un document .tex sauf que grâce à Python, nous pouvons faire faire le calcul par l'ordinateur et ainsi gagner beaucoup de temps et d'efficacité.

Cependant, pour utiliser ce package, il faut a priori un environnement LaTeX et un environnement Perl.

L'objectif de cette page est donc de présenter les différentes méthodes pour créer un document LaTeX via Pylatex.





Les titres

La première chose à comprendre avec ce package est que si nous avons un titre, il faut indenter le code python. Ce code reposant intégralement sur la synthaxe python, le contenu d'une partie doit être indenté et doit être désindenté si nous avons finis une partie.

Contrairement à LaTeX pur, avec Pylatex nous n'avons que trois niveaux de titre: section, subsection et subsubsection.

            

                # Le code Python
                
                from pylatex import Document, Section, Subsection, Subsubsection
                
                geometry_options = {"top=2cm, bottom=2cm, left=2cm, right=2cm"}
                doc = Document(geometry_options=geometry_options)

                with doc.create(Section("Titre de la section")):
                    # Le texte dans la section
                    with doc.create(Subsection("Titre de la subsection")):
                        # Le texte de la subsection
                        with doc.create(Subsubsection("Titre de la subsubsection")):
                        # Le texte de la subsubsection
                
                with doc.create(Section("Titre de la deuxième section")):
                    # Le texte dans la deuxième section
                
                doc.generate_pdf("full", clean_tex=False)




Ecrire du texte

Toute saisie de texte doit se faire via la ligne suivante:

            

                # Le code Python
                
                from pylatex import Document, Section
                
                geometry_options = {"top=2cm, bottom=2cm, left=2cm, right=2cm"}
                doc = Document(geometry_options=geometry_options)

                with doc.create(Section('Titre de la section')):
                    doc.append('Du texte dans la section')
                
                doc.generate_pdf('full', clean_tex=False)




Tracer un graphe

Pour tracer un graphe en pylatex, nous pouvons utiliser l'environnement Tikz:

            

                # Le code Python
                
                from pylatex import TikZ, Axis, Plot
                
                geometry_options = {"top=2cm, bottom=2cm, left=2cm, right=2cm"}
doc = Document(geometry_options=geometry_options)

                with doc.create(TikZ()):
                    plot_options = 'height=10cm, width=10cm, grid=major'
                    with doc.create(Axis(options=plot_options)) as plot:
                        plot.append(Plot(name='model', func='x^2'))
                    
                        coordinates = [
                            (-4.77778, 2027.60977),
                            (-3.55556, 347.84069),
                            (-2.33333, 22.58953),
                            (-1.11111, -493.50066),
                             (0.11111, 46.66082),
                             (1.33333, -205.56286),
                             (2.55556, -341.40638),
                             (3.77778, -1169.24780),
                             (5.00000, -3269.56775),
                    ]

                        plot.append(Plot(name='estimate', coordinates=coordinates))
                
                doc.generate_pdf('full', clean_tex=False)




Equation

            

                # Le code Python
                
                import numpy as np
                from pylatex import Math
                
                geometry_options = {"top=2cm, bottom=2cm, left=2cm, right=2cm"}
                doc = Document(geometry_options=geometry_options)

                a = np.array([[100, 10, 20]]).T
                M = np.matrix([[2, 3, 4],
                               [0, 0, 1],
                               [0, 0, 2]])
                               
                doc.append(Math(data=[Matrix(M), Matrix(a), '=', Matrix(M * a)]))
                doc.append(Math(data=["3*2 = 6]))
                
                doc.generate_pdf('full', clean_tex=False)
